day02

2021-06-22 关于实训

53个关键字 全部是小写
const goto 原来不是叫保留字 现在是了在新的jdk版本中
标识符 —-在java中可以自定义的名称
组成范围

  1. 各国常见文字
    1. 数字 0-9
  2. 特殊的符号 _和 $（慎用—内部类会使用）
    命名规则
    1.不能出现关键字
    2.不能以数字开头
    在组成范围内的内容且遵守命名规则的是符合ja0 va语法的标识符（合法）
    驼峰命令法（在标识符合法的基础上）
    1.类名/接口名——XxxYyyZzz
    2.方法名/变量名——-xxxYyyZzz
    3.包名—— 包是用来区分同名类的 （单级文件夹—-xxx cn。 多级文件夹——xxx.yyy.zzz。 tedu.cn.day01
    4.见名知意
    注释 /* */ 多行注释 单行注释 //
    文档注释 /** */ 可以通过javadoc命令生成解释说明的文档
    常量
    程序运行过程中不发生改变的值
    分类
    字面量常值 整数常量 1000 1839
                    小数常量 1.1 3.3 
                字符串常量 “12” “guii”
                字符常量“u”
                布尔常量 true false 
                空常量 null
    整数表示方式
    二进制 由0和1表示 以0b开头的数据就是二进制数据 0b 1000
    八进制 由0-7表示 以0开头的数据就是八进制数据
    十进制
    十六进制
    其他进制转换成十进制
    总结规律：项乘以基数为底的权次幂之和
    十进制转换为其他进制
    总结规律
    除基取余 直到商为0 余数翻转
    有符号的数
    计算机上所有的数据都在底层都是以二进制数据的补码形式来表示的
    数据类型 ：限制数据变化范围
    变量名：区分其他变量
    初始化：给变量赋值
